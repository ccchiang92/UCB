Box Office Analytics Visualization

This is a simply website that display some example data visualizations for our first project for UC Berkeley extension's data analytics boot camp.
The project is about finding what are important factors in making a financially successful movie, we chose this topic because the subjects is relatable, and there are lots of easily accessible, public data, also we can apply the same process for similar analysis to any product/category with sales data, amazon review score, etc.

We grabbed various data such as review ratings, opening box office, production budget, for the top 100 grossing movies of each year, between the years of 2000-2017, so 1800 movies total.
Then we clean up the data, adjusted all numbers for inflation and the plotted many different plots. This website only has a handful of selection, if you are interested please visit the github page for more details.
